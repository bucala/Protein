# ProteinDB

SK/CZ databáza proteínových doplnkov v jednom HTML súbore.

## Funkcie
- Vyhľadávanie, filtrovanie a triedenie produktov.
- Detail produktu v modálnom okne.
- Statické obrázky bez externého AI fetchu.

## Spustenie

Otvorte `protein-db.html` v prehliadači alebo nasadte cez GitHub Pages.
